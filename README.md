# Elm Forecast Weather App

A simple web app written in Elm.

### Install:
```
$ npm install
$ elm-package install
```

### Serve locally:
```
$ npm start
```
* Access app at `http://localhost:8080/`
* Or deployed at [surge](http://elm-forecast.surge.sh/)
